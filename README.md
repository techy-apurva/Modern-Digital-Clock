# Modern-Digital-Clock
A visually modern and fully responsive Digital Clock Web Application built using HTML, CSS, and JavaScript. This displays the current time in hours, minutes, and seconds, updating in real time. It includes a Light/Dark mode toggle that dynamically changes the theme and background, providing an engaging and user-friendly interface.


Features

Real-time clock with automatic updates every second

Light/Dark mode theme toggle with dynamic background change

Clean, modern, and visually appealing user interface

Fully responsive design for desktop, tablet, and mobile screens

Smooth UI interactions for enhanced user experience



Technologies Used

HTML5 – Semantic structure of the application

CSS3 – Styling, themes, and responsive design

JavaScript – Real-time clock functionality and DOM manipulation




Future Enhancements

Add alarm and timer functionalities

Option to customize fonts and colors

Add multiple clock zones for different countries

Integrate animations and transitions for smoother UI
